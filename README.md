# 🚀 웹 앱 실행 방법

1. **모든 코드를 다운로드한 상태에서 Terminal을 열기**
  

2. **프로젝트 디렉토리로 이동**
  `cd interactive-map`

3. **필요한 dependencies 설치**
  `npm install`
4. **개발 서버 시작**
  `npm run dev`

- npm run dev 하고 나서
Local: http://localhost:5173/

링크를 누르면 웹페이지로 리디렉팅 될 것이다.
